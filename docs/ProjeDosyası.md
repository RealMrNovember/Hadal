TEKNİK MİMARİ (v3 — Production MMO Hardened)

Engine: Unity 6 LTS · URP · **VContainer** · **Protobuf** · **HADAL.Shared** · PatchService

**Client = View + Input + Optimistic Prediction** (simülasyon sunucuda)

```
Input → ClientPredictionSystem (optimistic visual)
     → CommandDispatcher (Protobuf)
     → Gateway → Game Server (HADAL.Shared validation)
     → PostgreSQL + StateDelta (Protobuf)
     → Reconciliation / RollbackAnimator
     → VisualStateCache → Translator → LocalEventBus → UI
```

**Wire:** Gameplay WebSocket = Protobuf ONLY (JSON yasak)  
**Shared:** HADAL.Shared — commands, DTOs, enums, formulas (tek kaynak)  
**Sunucu:** 31.40.199.47 · hadal.cicibyte.com · [HADAL_SERVER_INFRASTRUCTURE.md](./HADAL_SERVER_INFRASTRUCTURE.md)

**YASAK:** local gameplay save · JSON replication · ServiceLocator · static singleton · Network→UI direct

**Bootstrap:** Boot → VContainer → Patch → Addressables → Login → Schema handshake → StateSnapshot → MainMenu

Detay: [19_HADAL_Shared_Protocol_And_Serialization.md](./19_HADAL_Shared_Protocol_And_Serialization.md)
